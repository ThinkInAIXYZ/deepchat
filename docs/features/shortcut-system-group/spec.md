# Shortcut System Group

## User Story

As a settings user, I want Shortcut settings to appear under the System group, so the sidebar organization matches where I expect app-level controls to live.

## Acceptance Criteria

- Shortcut settings appears in the System sidebar group.
- The Shortcut route, label, and page behavior remain unchanged.

## Non-goals

- No changes to shortcut editing behavior.
- No changes to route paths or i18n keys.
