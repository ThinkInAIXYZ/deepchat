# Sync & Upgrade Specification

**Status**: Draft
**Created**: 2026-01-19
**Owner**: Eric

## 背景与目标
同步与升级影响数据安全与稳定性。目标是界定同步流程、升级策略与 UI 通知边界。

## 范围（In Scope）
- 同步触发、状态展示与冲突提示。
- 升级检查、下载与提示流程。
- 数据导入/导出的入口管理。

## 非目标（Out of Scope）
- 具体同步协议与存储实现细节。
- Conversation 与消息主流程。

## 用户故事
- 用户可以看到同步状态并处理冲突。
- 用户可控升级策略与提示频率。

## 验收标准
- 同步/升级流程与业务域解耦。
- 所有副作用通过适配层触发。

## 约束与假设
- 同步实现由主进程或外部服务提供。

## 开放问题
无。
