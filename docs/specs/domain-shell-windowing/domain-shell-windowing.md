# Shell & Windowing Specification

**Status**: Draft
**Created**: 2026-01-19
**Owner**: Eric

## 背景与目标
多入口与窗口管理影响应用初始化与资源复用。目标是明确 shell 入口与窗口状态管理边界。

## 范围（In Scope）
- 多入口装配与共享初始化流程。
- 窗口状态（最小化、置顶、尺寸）管理。
- 与系统交互的状态展示。

## 非目标（Out of Scope）
- Conversation 与消息逻辑。
- 设置项的具体管理。

## 用户故事
- 用户在多个窗口间切换时状态一致。
- 启动流程可复用并易于维护。

## 验收标准
- 窗口管理逻辑集中，UI 只调用适配层。
- 入口初始化流程最小化且可复用。

## 约束与假设
- 窗口操作通过主进程或预加载桥接。

## 开放问题
无。
