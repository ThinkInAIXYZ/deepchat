# Navigation & Layout Specification

**Status**: Draft
**Created**: 2026-01-19
**Owner**: Eric

## 背景与目标
导航与布局是多入口 UI 的骨架层，当前初始化与状态分散。目标是明确布局状态与路由行为边界。

## 范围（In Scope）
- 主布局结构与可见区域状态。
- 导航行为（侧边栏、标签页、路由切换）。
- 导航相关的 UI 状态持久化。

## 非目标（Out of Scope）
- Conversation 与消息业务逻辑。
- 具体设置项与模型配置。

## 用户故事
- 用户在不同入口间导航时布局状态保持一致。
- 侧边栏与导航组件的状态可复用。

## 验收标准
- 导航状态与业务状态分离。
- 入口之间共享最小的装配逻辑。

## 约束与假设
- 路由层不直接依赖 presenter。

## 开放问题
无。
